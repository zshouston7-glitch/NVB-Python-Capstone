# 🍰 Interactive Recipe Manager 🧁

## What it does
The Interactive Recipe Manager is a cozy, easy-to-use Python application created to help home bakers and cooks organize their favorite recipes in one sweet spot! 📖✨ It solves the hassle of tracking down scattered recipes, helps prevent food waste by searching for dishes based on yummy ingredients already in your fridge 🫐, and takes away all the scary kitchen math by automatically scaling ingredient quantities whenever you want to cook for a crowd! 🍪

## How to run it
1. Open the notebook in Google Colab
2. Upload the dataset file if required (Option A projects)
3. Run all cells from top to bottom

## Skills demonstrated
- 🧺 **Nested Dictionaries & Lists:** Organized complex recipe data with ingredients, steps, and servings
- 🔄 **Indefinite `while` Loops:** Created a persistent, friendly menu loop with `if`/`elif`/`else` choices
- 💡 **Custom Functions & Docstrings:** Built clean, reusable functions with standard Python documentation
- 🎨 **Clean Code Principles:** Used clear variable names and added emojis for visually appealing terminal output

## Key finding (Option A) / What it produces (Options B, C, D)
A charming, interactive terminal app pre-loaded with delicious sweet treats (Chocolate Chip Cookies, Blueberry Muffins, and Blackberry Cobbler!) that lets users seamlessly add, search, view, and scale recipes.
